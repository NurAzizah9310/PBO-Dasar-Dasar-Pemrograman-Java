Dasar-Dasar Pemrograman Java - Pertemuan 2
Repositori ini berisi implementasi algoritma dasar Java sebagai bagian dari tugas mata kuliah IK222 - Pemrograman Berorientasi Objek.
Tugas ini mencakup penggunaan Input/Output, Operator Aritmatika, Operator Logika, dan Tipe Data.
Topik Pembelajaran Berdasarkan materi perkuliahan:
Mengimplementasikan konsep:I-P-O (Input-Proses-Output): Menggunakan Scanner untuk input dan System.out untuk output.
Tipe Data: Penggunaan int, double, boolean, dan char.
Operator: Implementasi operator Aritmatika, Logika, dan Perbandingan.
Implementasi Kode1.
Operasi Aritmatika Tiga Bilangan Algoritma ini menerima tiga input bilangan bulat dan menampilkan hasil operasi penjumlahan (+), pengurangan (-), perkalian (*), dan pembagian (/).
Konsep: Menggunakan Scanner.nextInt() dan operator aritmatika. 
Ekspresi Logika "Beli Gorengan" Algoritma untuk menentukan kondisi keluar membeli gorengan berdasarkan jumlah uang dan kondisi cuaca.
Ekspresi: (uang > 5000) && (!hujan) Konsep: Menggunakan Operator Logika AND (&&) dan NOT (!) , serta Operator Perbandingan (>).
Penjumlahan Kode ASCII Algoritma untuk menjumlahkan nilai ASCII dari karakter '1' dan '2'.Konsep: Di Java, tipe data char dapat menyimpan nilai ASCII. Penjumlahan dua karakter akan menghasilkan nilai integer dari gabungan kode ASCII keduanya.
